
# 🗄️ Projet SQL — Base de données pour un revendeur de thé

Projet de **modélisation et d’implémentation SQL** pour un revendeur de thé.

L’objectif est de concevoir une base de données relationnelle permettant de suivre les clients, les produits, les commandes et les ventes, puis de produire des indicateurs utiles au pilotage commercial.

---

## 🎯 Objectif du projet

Ce projet vise à construire une base SQL complète à partir d’un besoin métier.

La base permet de :

- stocker les informations clients ;
- gérer les produits vendus ;
- suivre les commandes ;
- analyser les ventes ;
- créer des vues SQL pour faciliter le reporting.

---

## 🧠 Compétences Data Engineering travaillées

Ce projet met en avant des bases importantes du métier de Data Engineer :

- modélisation relationnelle ;
- création de tables ;
- définition de clés primaires et étrangères ;
- insertion de données ;
- écriture de requêtes SQL ;
- création de vues ;
- préparation de données pour l’analyse.

---

## 🛠️ Technologies utilisées

- SQL
- Modélisation relationnelle
- Schéma de base de données
- Contraintes d’intégrité
- Vues SQL
